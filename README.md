# v42 website

Live site: https://v42.space

Requires Node.js 16+.

To setup:

```
npm i 
npm i -g yarn
```

Run the dev server at http://localhost:53000:

```
yarn dev
```

To build the final product:

```
yarn build
```

To publish to live site:

```
yarn pub
```
